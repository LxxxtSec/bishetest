# 课题名称
无人机交通巡检系统操作安全审计平台

# 所属专业
网络空间安全学院（密码学院）

# 指导教师
周腾

# 选题类型
工程设计类

# 课题简介
针对无人机交通巡检系统的使用过程，设计并开发一个专注于操作安全审计的Web平台。该平台详细记录管理员、操作员对系统的关键操作（如用户权限变更、无人机任务指令下达、无人机任务指令下发、数据访问记录），并提供审计日志的查询、分析和报表生产功能。

# 课题要求
设计并实现系统操作日志的数据结构，确保关键操作信息（操作人、时间、动作、对象）的完整记录。
开发审计日志的查询、导出功能，支持按时间、操作类型、用户等多条件筛选。
实现可疑操作行为（如频繁登陆失败、非工作时间敏感操作）的简单规则检测与告警提示。
保证审计日志本身的安全性，防止被非法篡改或删除。

# 研究重点
信息安全审计在无人机交通巡检业务流程中的关键点分析；基于数据库触发器的日志自动记录技术；审计日志防篡改技术方案（如哈希校验）。
